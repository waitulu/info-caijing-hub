# info-caijing-hub

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面内容涵盖财经、信息整理等主题，以静态文件形式存放，方便浏览与分发。

所有页面均为独立 HTML 文件，不依赖特定域名或外部服务，可直接在浏览器中打开查看。

## 目录说明

```
info-caijing-hub/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面使用的资源文件（如样式、图片等）
├── README.md       # 本说明文件
└── index.html      # 可选：页面索引或导航页
```

- **pages/**：每个 HTML 文件代表一个独立的归档页面，文件命名尽量体现内容主题。
- **assets/**：存放页面引用的静态资源，如 CSS、JavaScript、图片等。
- **index.html**：如存在，可作为页面列表或导航入口。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件。
3. 如有 `index.html`，也可从首页导航进入各页面。
